# RVSP-Ride-to-CRC (Church Rides App)

This is a route-based ride coordination application connecting church members with volunteer drivers.
Built with Next.js, Supabase, Tailwind CSS, Leaflet, and Map Routing.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
